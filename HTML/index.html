GitHub es una plataforma en la nube que permite a los desarrolladores crear, almacenar, compartir y trabajar en proyectos de software. Entre sus funciones se encuentran:

  • Almacenar, descargar y modificar versiones de aplicaciones o herramientas
  • Compartir el trabajo con otros usuarios
  • Seguir y administrar los cambios en el código
  • Recibir sugerencias para mejorar el código
  • Trabajar en proyectos compartidos
  • Asignar tareas y administrar estados de tareas
  • Informar problemas con el código

Comandos principales de GitHub

----------------------------------------------------------------------------------------------

Configurar usuario en Git:

  • git config --global user.name "Tu Nombre"
  • git config --global user.email "tuemail@example.com"

Configura el nombre y correo para los commits.

----------------------------------------------------------------------------------------------

Inicializar un repositorio:

  • git init

Crea un nuevo repositorio Git en la carpeta actual.

----------------------------------------------------------------------------------------------

Vincular repositorio con GitHub:

  • git remote add origin URL-del-repositorio

Conecta el repositorio local con uno remoto en GitHub.

----------------------------------------------------------------------------------------------

Ver estado del repositorio:

  • git status

Muestra cambios pendientes y archivos sin seguimiento.

----------------------------------------------------------------------------------------------

Agregar archivos al commit:

  • git add .

Agrega todos los cambios al área de preparación.

----------------------------------------------------------------------------------------------

Realizar un commit:

  • git commit -m "Mensaje del commit"

Guarda los cambios en el historial de Git.

----------------------------------------------------------------------------------------------

Subir cambios a GitHub:

  • git push origin main

Envía los commits al repositorio remoto en la rama main.

----------------------------------------------------------------------------------------------

Clonar un repositorio de GitHub:

  • git clone URL-del-repositorio

Descarga una copia exacta del repositorio remoto.

----------------------------------------------------------------------------------------------

Actualizar el repositorio local:

  • git pull origin main

Trae los últimos cambios del remoto a la máquina local.

----------------------------------------------------------------------------------------------

Crear una nueva branch:

  • git branch nombre-de-la-branch

Genera una nueva rama para desarrollo paralelo.

----------------------------------------------------------------------------------------------

Cambiar de branch:

  • git checkout nombre-de-la-branch

Cambia a la rama seleccionada.

----------------------------------------------------------------------------------------------

Fusionar branches:

  • git checkout main
  • git merge nombre-de-la-branch

Une los cambios de una rama en main.

----------------------------------------------------------------------------------------------

Eliminar una branch:

  • git branch -d nombre-de-la-branch

Elimina una rama localmente.

----------------------------------------------------------------------------------------------

Eliminar una branch remota:

  • git push origin --delete nombre-de-la-branch

Borra una rama en el repositorio remoto.

----------------------------------------------------------------------------------------------

Ver repositorios remotos configurados:

  • git remote -v

Muestra los repositorios remotos vinculados.

----------------------------------------------------------------------------------------------

Actualizar y limpiar branches remotas eliminadas:

  • git fetch --prune

Sincroniza el repositorio y borra referencias a branches eliminadas.

----------------------------------------------------------------------------------------------
